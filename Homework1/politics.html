Assuming Stuxnet was created by the US government--it had some political motivations behind it. As mentioned previously, the average user wasn't affected by stuxnet at all besides being a vector for it to spread. Stuxnet was created as a nonviolent way to shut down Iran's nuclear program. It would target a very specific system configuration that indicated the host computer was being used to control centrifuges that are used to refine uranium. If such a computer existed, the worm would exploit a bug in the micro-controller for the centrifuge and quietly takes over its administrative processes. Stuxnet will very slowly change settings in the centrifuge to eventually cause the enrichment process to fail, all while remaining undetected because it hides itself so well. It causes the controller for the centrifuge to lock down so nobody can stop it and it also causes the controller for it not to report any problems or warnings. All of these things combined would effectively destroy a nation's nuclear program. In this particular case: Iran.
